System overview

Short description

This aluminum curtain track is designed as a base system for matched carriers, connectors, end caps, and mounting brackets. It can be prepared in standard or custom lengths for wholesale and project orders.

Ceiling mount Wall mount Cut-to-length OEM packaging
  • Extruded profile for stable shape and consistent carrier movement
  • Compatible with standard carriers, rollers, end caps, and mounting brackets
  • Suitable for ceiling mount or wall mount installation depending on bracket choice
  • Available in raw, powder coated, or anodized finish options
