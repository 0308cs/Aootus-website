System overview

Short description

This aluminum track system is designed for smooth running performance, practical installation, and matched accessory supply. It can be prepared in standard or custom lengths for wholesale and project orders.

Ceiling mount Wall mount Cut-to-length OEM packaging
  • Extruded profile for stable shape and consistent carrier movement
  • Compatible with standard carriers, end caps, and mounting brackets
  • Suitable for light to medium curtain load in repeated project use
  • Available in raw, powder coated, or anodized finish options
